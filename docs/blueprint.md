# **App Name**: MedStock Tracker

## Core Features:

- Medicine Entry: Add new medicines with details like name, initial stock quantity, and dosage per day.
- Dosage Scheduling: Set up daily dosage schedules for each medicine, including time and amount.
- Automated Stock Depletion: Automatically decrease stock levels based on the scheduled dosages.
- Current Stock Display: Show current quantities of each medicine in an easily readable format.
- Stock End Date Prediction: Predict and display the estimated end date and time for each medicine based on the stock and dosage.

## Style Guidelines:

- Primary color: Dark blue (#2C3E50) to evoke trust and stability, suitable for a medical context.
- Background color: Light grey (#ECF0F1), creating a clean, high-contrast interface with a subtle blue hue.
- Accent color: Electric blue (#7DF9FF), drawing attention to essential interactive elements and stock level warnings.
- Body and headline font: 'Inter', a grotesque sans-serif with a neutral, modern look.
- Use flat, line-style icons for medicine types, dosage schedules, and stock levels.
- Employ a grid-based layout to present medicine data in a structured, clear manner.
